# Lectrax platform-admin deployment

This folder is generated by `npm run export:admin` from the main Lectrax repository.

## Architecture

| Deployment | Domain example | Env |
|------------|----------------|-----|
| Main app (lecturers + students) | `https://lectrax.app` | `NEXT_PUBLIC_ADMIN_APP_URL=https://admin.lectrax.app` |
| Admin app (platform admin only) | `https://admin.lectrax.app` | `NEXT_PUBLIC_DEPLOYMENT_TARGET=admin` |

Both apps share the **same Supabase project**. Platform admins sign in only on the admin domain.

## Generate / refresh export

From the main repository root:

```bash
npm run export:admin
```

## Push to a separate GitHub repository

```bash
cd deploy/lectrax-admin
git init
git add .
git commit -m "Lectrax platform admin export"
git branch -M main
git remote add origin https://github.com/YOUR_ORG/lectrax-admin.git
git push -u origin main
```

Re-export and push updates:

```bash
npm run export:admin
cd deploy/lectrax-admin
git add .
git commit -m "Sync admin export from main app"
git push
```

## Admin app environment

Copy `.env.example` to `.env.local` in `deploy/lectrax-admin/`:

```env
NEXT_PUBLIC_DEPLOYMENT_TARGET=admin
NEXT_PUBLIC_APP_URL=https://admin.lectrax.app
NEXT_PUBLIC_MAIN_APP_URL=https://lectrax.app
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
SUPABASE_SERVICE_ROLE_KEY=...
```

## Main app environment

On the lecturer/student deployment, set:

```env
NEXT_PUBLIC_APP_URL=https://lectrax.app
NEXT_PUBLIC_ADMIN_APP_URL=https://admin.lectrax.app
```

When `NEXT_PUBLIC_ADMIN_APP_URL` is set, `/admin` routes on the main app redirect to the admin domain.

## Supabase Auth

Add both URLs to **Authentication → URL configuration**:

- Site URL: your main app URL
- Redirect URLs:
  - `https://lectrax.app/auth/callback`
  - `https://admin.lectrax.app/auth/callback`

## Local development

Terminal 1 — main app:

```bash
npm run dev
```

Terminal 2 — admin app:

```bash
npm run export:admin
cd deploy/lectrax-admin
npm install
cp .env.example .env.local
npm run dev
```

Main app: http://localhost:3000  
Admin app: http://localhost:3001
